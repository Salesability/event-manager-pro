## Codex Adversarial Review (pass 2)

### Critical / High

### Medium

1. **M2 still partially open — malformed range row fails open.** `emitVariable` guards only run when *both* `unitPriceMin` AND `unitPriceMax` are non-null. The schema allows null bounds on a `range`-typed row, so a hand-edited or future-seeded malformed row would silently accept arbitrary record-retrieval amounts. Fix: fail closed for any `range` row whose bounds aren't finite numbers.

2. **NEW — tax rounding inconsistency between `moneyString` and `roundCents`.** `setQuoteTax` writes `tax.toFixed(2)` directly; `createQuote` / `setQuoteInputs` route tax through `computeQuote.roundCents()`. For inputs like `2.675`, the two paths can produce different cents depending on IEEE-754 representation. Fix: reject tax with > 2 decimal places at parse time, or normalize tax through one shared cents function before persisting.
