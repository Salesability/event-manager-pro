### False positives Codex considered and ruled out

- `src/features/quotes/queries.ts:97-125` — `innerJoin(dealers, eq(dealers.id, quotes.dealerId))` has no `dealers.archivedAt IS NULL` predicate. `dealers.archivedAt` is selected into the projection so archived dealers' quotes still surface (intended for read-only edit-mode hydration); only orphaned-dealer rows would be dropped, and `quotes.dealerId` is a `RESTRICT` FK per `src/lib/db/schema/quotes.ts:51-53`.
- `src/features/quotes/queries.ts:55-87` — `inputs` cast from `unknown` to `QuoteInputs` is acceptable; the write path canonicalizes the stored shape via `parseQuoteInputs` at `src/features/quotes/actions.ts:91-121`.

- Repo-wide call-site search for `import-from-sheets` / `quoteValidDays` / `depositPct` / `taxPct` / `travel` / `quoteNotes` — no live consumers depend on the dropped campaign columns.
