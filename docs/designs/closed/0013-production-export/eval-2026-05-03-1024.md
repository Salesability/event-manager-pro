## Codex Adversarial Review

```
**Critical**

None.

**High**

None.

**Medium**

CSV formula injection is not mitigated: `src/app/(app)/production/export/route.ts:33-47` and `:66-67`. Quoting handles CSV syntax, but Excel/Sheets can still treat cells starting with `=`, `+`, `-`, or `@` as formulas; dealer/contact/notes are user-influenced fields, so this is a real low-friction risk.
Suggested fix: Prefix formula-looking cell values with `'` or `\t` inside `csvCell`, before quote escaping.

**Low**

Export materializes the full campaign table before filtering: `src/app/(app)/production/export/route.ts:28` calls `filterCampaigns(await loadCampaigns(), ...)`; `src/features/schedule/queries.ts:329-361` has no `where`/`limit`. Matches the page's pattern, but the export builds `rows`, `lines`, and the full CSV string in memory.
Suggested fix: Push status/date/search filters into Drizzle and stream CSV rows if data can grow materially.

Toasts and open dialogs can leak into print output: `src/app/(app)/layout.tsx:16` renders `<Toaster />`; `src/components/ui/dialog.tsx:45,65` uses `fixed inset-0 z-40/z-50`. The new `print:hidden` covers header/filters/row actions, but global portals are not hidden when visible at print time.
Suggested fix: Add print-hide styling to the toaster/dialog portal/backdrop/panel classes, or add a global `@media print` rule for known overlay containers.

Export link can lag the visible search input: `src/app/(app)/production/production-filters.tsx:44-58` debounces `setTimeout(() => pushParams({ q: value }), 250)` while `exportHref` uses current `params.toString()`. If a user types then immediately clicks Export CSV, the href reflects the previous query.
Suggested fix: Build `exportHref` from local `q` plus current `status`/`cancelled`, or flush the pending debounce before export.

**Not An Issue**

- Auth bypass: `src/proxy.ts:7-10` matches `/production/export`; `src/lib/supabase/middleware.ts:5,32-36` redirects non-public unauthenticated paths.
- Header injection: filename comes from `todayIso()` (`filter.ts:9-11`), numeric only.
- CSV newlines: `route.ts:54` joins with `\r\n` and `csvCell` quotes/doubles quotes — embedded newlines survive.
- Filter divergence: page (`page.tsx:31`) and route (`route.ts:28`) both call shared `filterCampaigns` (`filter.ts:14-32`).
- Route group matching: `(app)` is not in the URL — `/production/export` is matched by middleware.
- Status date comparisons: ISO `YYYY-MM-DD` string comparisons in `filter.ts:18-19` and `route.ts:77-78` are chronological for valid date-only strings.
```

## Action Required

### Must Fix (blockers)
None.

### Should Fix (Codex medium)
1. **CSV formula injection** in `src/app/(app)/production/export/route.ts:66-67` — prefix cells beginning with `=` / `+` / `-` / `@` with a leading `'` (or `\t`) inside `csvCell` before the quote-and-escape step. Dealer name, contact, and notes are user-influenced; a malicious or accidental `=HYPERLINK(…)` in a dealer name would execute when the CSV opens in Excel.

### Consider (Codex low / smoke note)
1. Print-mode leak for `Toaster` / `Dialog` portals — add `@media print { .sonner-toast, [data-sonner-toaster], [role="dialog"] { display: none !important; } }` to `globals.css` so a stray open toast or dialog doesn't print.
2. Debounce vs export link race — build `exportHref` from the local `q` state plus URL `status`/`cancelled` rather than `params.toString()`, so a freshly-typed search includes in the export.
3. PDF column overflow — table still clips right-most columns even with `print:text-[10px]`. Either tighten column widths under `@media print` or add `@page { size: A4 landscape; }`.
4. `loadCampaigns()` has no LIMIT — fine at 42 campaigns, worth revisiting if the dataset grows past a few thousand.
