**Resolution:** Split the explanation. Admin-page affordances are double-gated (page + `<Can>`). The Calendar `Block Date` is layout-gated at the page level, but the `<Can capability="availability:edit">` matches the action's capability so the affordance is hidden for any role that wouldn't pass the action.
